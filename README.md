<h1 align="center"> 3DTile-Viewer </h1>



## What it is?

**3DTile-Viewer** is a tool for viewing 3dtile data in the vscode editor. The data formats you can view are.json,.pnts,.b3dm,.i3dm,.cmpt.

4. Enjoy!
